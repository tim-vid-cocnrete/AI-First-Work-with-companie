# Справочник статусов Лидов
**Описание**: Интерпретация статусов лидов (Qualified, Disqualified, Refusals) для аналитики.
**Категория**: Marketing

> [!IMPORTANT]
> **Источник**: `Docs/SalesAndMarketing/User Research/lead_statuses_interpretation.md`

На основе анализа файла `Leads June-October.csv` были выявлены следующие статусы лидов.

## 1. Успешная квалификация (Quals)
Лиды, которые прошли первичный отбор и перешли на следующий этап (выбор формы оплаты).

| Статус | Интерпретация |
| :--- | :--- |
| **Квалифицирован** | Лид подтвердил интерес, соответствует критериям обучения и готов обсуждать условия (переходит в воронку Hybrid или Prepayment). |
| **Уже устроился на работу** | Лид нашел работу самостоятельно (актуально для тех, кто искал гарантии трудоустройства). |
| **Исключительно ISA** | Лид жестко ориентирован только на модель ISA (оплата после трудоустройства). |
| **Повторная квалификация** | Лид вернулся в воронку или требует обновления данных квалификации. |

## 2. Отбракованные (Disqualified)
Лиды, с которыми контакт состоялся, но они не могут быть приняты по объективным причинам.

| Статус | Интерпретация |
| :--- | :--- |
| **Не подходит для поступления** | Лид не прошел квалификацию по формальным признакам (возраст, гражданство, отсутствие ПК, технические ограничения и т.д.). |
| **Невменяемый** | Неадекватное поведение при общении, агрессия или спам. |
| **Дубли** | Технический дубль заявки (повторная регистрация существующего лида). Удаляется из обработки. |
| **Не оставлял заявку** | Человек утверждает, что не заполнял форму (ошибка ввода номера, миссклик или бот). |
| **Неверный телефон и email** | Контактные данные некорректны (опечатка, несуществующий номер/почта). |

## 3. Невыход на связь (No Contact)
Лиды, до которых менеджеры не смогли достучаться.

| Статус | Интерпретация |
| :--- | :--- |
| **Тотальный недозвон** | Менеджер совершил регламентированное количество попыток звонков, но трубку никто не взял. |
| **Не вышел на связь TG/WA/VK** | Менеджер пытался связаться через мессенджеры (Telegram, WhatsApp, VK), но сообщения не прочитаны или проигнорированы. |
| **Пропал после квалификации** | Лид прошел первичный отбор, но перестал выходить на связь перед следующим этапом. |

## 4. Отказ и Возражения (Refusals)
Лиды, с которыми контакт состоялся, но сделка сорвалась.

| Статус | Интерпретация |
| :--- | :--- |
| **Отказался после консульт/не продал** | Менеджер провел диалог, но не смог убедить клиента. Клиент отказался продолжать общение. |
| **Передумал** | Клиент выразил отказ до полноценной презентации или в самом начале квалификации. |
| **Нет денег** | Специфический отказ по финансовым причинам. |
| **Выбрал конкурентов** | Лид решил обучаться в другой школе или выбрал другой способ обучения. |
| **Нет времени на обучение** | Отказ по причине нехватки времени (работа, учеба, личные обстоятельства). |
