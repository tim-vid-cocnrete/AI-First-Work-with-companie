# Структура Файлов Кандидата

Каждая папка кандидата должна следовать этой структуре:

```
Candidate-[Имя]/
├── CV.md                         # Резюме/CV кандидата
├── Fathom-Screening-Interview.md # Заметки с первичного скрининг интервью
├── Fathom-Technical-Interview.md # Заметки с технического интервью
├── AI-Notes.md                   # AI-сгенерированный анализ и инсайты
├── Screening-Evaluation.md       # Оценка с этапа скрининга
├── Evaluation.md                 # Комплексная оценка
└── Final-Score.md                # Финальный скоринг и решение
```

## Создание Новой Папки Кандидата

Чтобы создать новую папку кандидата с правильной структурой:

1. Скопируйте папку Template
2. Переименуйте ее в `Candidate-[Имя]` (например, `Candidate-Jane-Smith`)
3. Заполните релевантную информацию в каждом файле

## Описания Файлов

- **CV.md**: Содержит информацию из резюме кандидата
- **Fathom-Screening-Interview.md**: Саммари и заметки с начального скрининг интервью
- **Fathom-Technical-Interview.md**: Детальные заметки с технического ассессмент интервью
- **AI-Notes.md**: Автоматизированный анализ материалов кандидата
- **Screening-Evaluation.md**: Форма оценки, заполняемая после скрининг интервью
- **Evaluation.md**: Комплексная форма оценки с детальным скорингом
- **Final-Score.md**: Документ финального решения с общими баллами и рекомендацией по найму

## Использование Шаблона

Папка Template содержит пустые версии всех требуемых файлов с правильной структурой. Используйте их как отправную точку для каждого нового кандидата.
