# Contacts Actual - Active Fundraising Contacts

**File Purpose**: Quick reference for all active contacts in the fundraising pipeline. Format: compact keywords/tags separated by commas for instant association. Used by both agents and humans for:

- Quick context recall ("oh, that's the person who...")
- Finding relevant people by topics/connections
- Compiling follow-up task summaries
- Pre-call/meeting context preparation

**Cross-Reference**: For overall fundraising progress, news, and updates see [Fundraisng-Process-Status.md](Fundraisng-Process-Status.md)

**Entry Format**:

- **Name** (role/company) - keywords separated by commas, last contact date
- **Tasks from contact** → what they promised/should do
- **Tasks on our side** → what we need to do
- **Status** → current interaction stage
