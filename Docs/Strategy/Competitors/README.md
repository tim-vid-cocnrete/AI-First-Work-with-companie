# ⚔️ Конкурентный Анализ

> [!TIP]
> В этой папке храните детальные разборы ваших конкурентов.
> Примеры переноса в архив: `Docs/_Archive/Templates/Competitors/`

## Как анализировать конкурента?
1. Создайте файл `[название-конкурента]-analysis.md`.
2. Опишите их сильные и слабые стороны.
3. Добавьте ссылку в `competitive-landscape-summary.md` (если он есть).
