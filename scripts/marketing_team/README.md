
# AI-команда для маркетинговой стратегии

## Введение
Этот проект демонстрирует использование фреймворка CrewAI для автоматизации создания маркетинговой стратегии. CrewAI координирует работу автономных AI-агентов, позволяя им сотрудничать и эффективно выполнять сложные задачи.

## Фреймворк CrewAI
CrewAI разработан для облегчения сотрудничества ролевых AI-агентов. В этом примере агенты работают вместе над созданием комплексной маркетинговой стратегии и разработкой привлекательного маркетингового контента.

## Запуск скрипта
По умолчанию используется GPT-4o, поэтому у вас должен быть доступ к этой модели для запуска.

***Отказ от ответственности:** Скрипт использует gpt-4o, если вы не измените настройки на другую модель. Использование других моделей может привести к изменению стоимости.*

- **Настройка окружения**: Скопируйте `.env.example` и настройте переменные окружения для [OpenAI](https://platform.openai.com/api-keys) и других необходимых инструментов, например [Serper](serper.dev).
- **Установка зависимостей**: Запустите `poetry lock && poetry install`.
- **Настройка**: Измените `src/marketing_posts/main.py`, чтобы добавить пользовательские входные данные для ваших агентов и задач.
- **Дополнительная настройка**: Проверьте `src/marketing_posts/config/agents.yaml` для обновления агентов и `src/marketing_posts/config/tasks.yaml` для обновления задач.
- **Выполнение скрипта**: Запустите `poetry run marketing_posts` и введите данные вашего проекта.

## Детали и описание
- **Запуск скрипта**: Выполните `poetry run marketing_posts`. Скрипт будет использовать фреймворк CrewAI для генерации детальной маркетинговой стратегии.
- **Ключевые компоненты**:
  - `src/marketing_posts/main.py`: Главный файл скрипта.
  - `src/marketing_posts/crew.py`: Основной файл команды (crew), где агенты и задачи объединяются и выполняется основная логика.
  - `src/marketing_posts/config/agents.yaml`: Файл конфигурации для определения агентов.
  - `src/marketing_posts/config/tasks.yaml`: Файл конфигурации для определения задач.
  - `src/marketing_posts/tools`: Содержит классы инструментов, используемых агентами.
